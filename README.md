# CG-T4
Autorama 3D OpenGL using Bspline curves


Desenvolva um programa em C++, utilizando a API OpenGL, para controlar um carro sobre
uma pista de corrida semelhante a um autorama. A pista deve ser formada por curvas B-
Spline. O programa deve permitir que o usuário controle a velocidade e direção do carro
enquanto o carro se movimenta sobre ela.

Requisitos Mínimos
● A pista deve ser um circuito fechado, em 3D, com elevação variada.
● O usuário deve poder controlar a direção do carro usando as setas do teclado.
● A geração da pista pode ser estática – feita a mão – Apenas os pontos de controle e
grafo de controle.
● A geometria do carro deve ser 3D (pode ser um cubo). Pode-se escolher visão interna
ou externa do carro (gluLookAt).
● Permitir que o usuário visualize o vetor direção do carro. Crie um botão para ativar e
desativar essa funcionalidade.
● Implementar qualquer tipo de iluminação (no mínimo difusa)
● Programa expande a linha definida pelo usuário e gera a pista com largura específica.
● Permitir que o usuário gere a pista utilizando o mouse. Dica: você pode se basear
nesse vídeo: https://www.youtube.com/watch?v=1HCbe8yTHNw
